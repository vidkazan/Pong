# Pong
- Real time pong game with chat, channels and user profile
- Single-page application
- Final project of the Ecole 42 commom core curriculum

## Used technoloies
- TypeScript
- NestJS
- ReactJS
- PostgreSQL
- Prisma
- OAuth
- Google Authenticator
- JWT
